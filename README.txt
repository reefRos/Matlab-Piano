Playing prerecorded songs:
First, pick a song from the "songs" list and afterwords press the "PLAY" button.
The song might take a few seconds to play on the first time.

Rerding songs:
First tick the Record box and as long as that box is ticked, each note pressed will be saved to an array.
When you are ready to stop recording, just press "SAVE" and untick the "RECORD" box. The array will be
written to an excel file. Note that the delays between each notes will have to be entered manually to the excel file.
 
Looking for recorded songs:
Enter the songs name to the search box including .xlsx ending and press "PLAY CUSTOM SONG" button
